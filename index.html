Custom FI Payment Solution

Designed and developed custom solutions ZNS_FI_INPAY and ZNS_FI_OTPAY to manage incoming/outgoing payment data flows. Implemented ALV reports for strict data verification, ensuring robust integrity and compliance for financial transactions.

MM Module Inventory Tracking ALV Report

역할: 풀스택 ABAP 개발자 Role: Full-stack ABAP Developer

자재관리(MM) 모듈의 실시간 재고 추적을 위한 커스텀 ALV 리포트입니다. MARA, MARC, MARD 등 표준 테이블 데이터를 결합하여 재고 현황을 모니터링하며, 대용량 데이터 조회 시 발생하는 성능 병목 현상을 해결하기 위해 이진 탐색(Binary Search) 성능 튜닝을 수행했습니다.

Developed a custom ALV report to track real-time inventory levels, combining data from standard SAP tables like MARA, MARC, and MARD. Overcame query bottlenecks for large data volumes using Binary Search optimization.
